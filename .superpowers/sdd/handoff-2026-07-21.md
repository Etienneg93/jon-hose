## Art pipeline — hardened rules learned this session

- Generation: codex-image skill (`codex exec` + prompt FILE via stdin,
  UTF-8 — cp1252 em-dashes break it; recover misses from
  ~/.codex/generated_images/<session>/). Single-image generations ONLY —
  sheet slicing caused every debris class we fought.
- Processing (tmp/process_assman.py, tmp/process_air.py patterns):
  **4-connectivity for background floods** (8-conn slips diagonal outline
  seams), two-tone checker keying, opaque checker-slab kill =
  neutral-light comps TOUCHING transparency (outlined cloud/white cores
  never do), enclosed-pocket clears only when ringed ≥50% dark, cloud
  bands protected, color-bleed before LANCZOS, alpha ≥128 hard.
- Bakes: `tools/assman-bake.py` (masters → baked/ + _l variants; safe to
  re-run, never touches masters). NEVER re-run bakers over
  sprites/mook/* or sprites/fuse/walk* (hand-cleaned).
- Approval flow: generate → present → user verifies (memory:
  sprite-generation-workflow). The review board artifact:
  https://claude.ai/code/artifact/65196741-1fbb-4d1f-a2e4-344f36b300a0
  — update from ANY session by passing that URL as `url` to the Artifact
  tool; a copy of the built html is at tmp/air-act-gates-latest.html
  (base64-inlined, regenerate imagery from repo assets when stale).
